NCCN Guidelines Downloader

Nasıl Kullanılır?

Bu araç tamamen sunucusuz çalışır ve doğrudan GitHub Actions üzerinden tetiklenir.

  1. Aşağıdaki butona tıklayarak GitHub Actions sayfasına gidin.
  2. Sol taraftaki listeden "NCCN Guidelines Downloader" iş akışını seçin.
  3. "Run workflow" butonuna tıklayın.
  4. Açılan pencerede NCCN kullanıcı adınızı ve şifrenizi girerek işlemi başlatın.

İşlem tamamlandığında PDF dosyaları otomatik olarak bu depodaki downloads/ klasörüne eklenecektir.
